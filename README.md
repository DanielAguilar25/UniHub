# UniHub 🎓
A social campus web app for UTRGV students to find study partners, meet people, discover clubs, and stay up to date with campus events.

Overview
UniHub is built specifically for UTRGV students. Whether you need a study partner before an exam, want to find people with similar interests, or just want to know what's happening on campus — UniHub is your go-to place.
The app allows students to:

- Find study partners — connect with classmates by course or subject
- Meet people — discover other students with similar interests.
- Explore clubs — browse and join UTRGV student organizations.
- Stay updated — view and RSVP to campus events


# Features
UniHub follows a CRUD convention:

- ➕ Create — Register an account, build a profile, create groups, and post events
- 👁️ Read — Browse other students' profiles, view events, and read posts
- ✏️ Update — Edit your profile, update your posts and events
- 🗑️ Delete — Remove your posts, events, or groups




# Tech Stack

- Frontend: React + Vite (Javascript)
- Backend: Django (Python)
- Styling: HTML + CSS
- Database: SQLite



# How to Run the App
Prerequisites

- Node.js v18+
- Python 3.x

Terminal 1 (React) Frontend

- Clone Repository **_(git clone repository url)_**
- Navigate Into the Project **_(cd unihub)_**
- Install Depedencies **_(npm install)_**
- Start the Development **_Server (npm run dev)_**

The frontend will be running at http://localhost:5173

Terminal 2 (Django) Backend

- Naviate Into Backend Folder **_(cd unihub/backend)_**
- Install Django **_(pip install django)_**
- Run Migrations **_(python manage.py migrate)_**
- Start the Server **_(python manage.py runserver)_**

The backend will be running at http://localhost:8000

⚠️ Both terminals must be running at the same time for the app to work.
